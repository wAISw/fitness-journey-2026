# Fitness Journey 2026

**Цель:** Укрепление кора, коррекция гиперлордоза, возвращение в форму после года перерыва, постепенный минус жира при сохранении мышц.

## Данные
- Пол: Мужчина
- Возраст: 37 лет
- Рост: 183 см
- Текущий вес: 97 кг
- Формат: 4 дня в зале (Push / Pull / Upper / Lower) + 3 дня отдыха
- Тренировки: утром, в одиночку, строго 60 минут

## Особенности
Небольшой гиперлордоз — приоритет нейтральной пояснице и глубокому кору.

## Структура
- `program/` — актуальная программа тренировок
- `logs/training/` — отчёты по тренировкам (Hevy)
- `logs/nutrition/` — дневник питания
- `logs/notes.md` — заметки о самочувствии, пояснице, энергии, сне
- `progress/weight/` — фото и вес (еженедельно)
- `progress/measurements.md` — обхваты
- `archive/` — старые версии программы
