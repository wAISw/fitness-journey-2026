# CLAUDE.md — Инструкция для ИИ-помощника

**Репозиторий:** Мой фитнес-трекер 2026  
**Цель:** Укрепление кора, коррекция гиперлордоза, возвращение в форму после года перерыва, постепенный минус жира при сохранении мышц.

**Основные данные владельца:**
- Пол: Мужчина
- Возраст: 37 лет
- Рост: 183 см
- Текущий вес: 94,4 кг (обновлено 2026-04-14)
- Тренировочный формат: 4 дня в зале (Push / Pull / Upper / Lower) + 3 дня отдыха
- Особенности: небольшой гиперлордоз → приоритет нейтральной пояснице и глубокому кору
- Тренировки: утром, в одиночку, строго 60 минут
- Вес в логах: экспорт Hevy (`workout_data.csv`) — в формате приложения (на гантель/сторону); вес тела — `measurement_data.csv`

## Цели ИИ (Claude / Grok)
При каждом анализе отчётов ты **обязан**:
1. Давать честную, но поддерживающую обратную связь.
2. Искать закономерности (прогресс весов, самочувствие, питание, поясница).
3. Предлагать конкретные правки в программе или питании только при необходимости.
4. Следить за безопасностью поясницы и гиперлордозом.
5. Сохранять историю изменений программы.
