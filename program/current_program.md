# Актуальная программа тренировок

**Версия:** 1.2
**Дата:** 2026-03-30
**Формат:** Push / Pull / Upper / Lower, 4 дня в неделю, 60 минут
**Структура недели:** Пн — PUSH | Вт — PULL | Ср — ВЫХОДНОЙ | Чт — UPPER | Пт — LOWER | Сб–Вс — ВЫХОДНОЙ
**Тайминг:** 5 мин эллипс → 45–48 мин силовая → 5 мин сауна

---

## День 1 — PUSH (Грудь / Плечи / Трицепс + Кор)

| Упражнение | Подходы × Повторения | Примечания |
|---|---|---|
| Жим гантелей лёжа на горизонтальной скамье | 4×10–12 | Нейтральная поясница, без прогиба |
| Жим гантелей сидя | 3×10 | Спина плотно прижата к спинке |
| Разведения гантелей в стороны стоя | 3×12–15 | Лёгкий вес, локти чуть согнуты |
| Французский жим одной гантели лёжа | 3×12 | Или в кроссовере |
| Dead Bug с лёгкими гантелями | 3×10/сторону | С дыханием диафрагмой |
| Plank на предплечьях | 3×40–60 сек | Дышать диафрагмой, не задерживать дыхание |

---

## День 2 — PULL (Спина / Бицепс / Задние дельты + Кор)

| Упражнение | Подходы × Повторения | Примечания |
|---|---|---|
| Тяга верхнего блока широким хватом к груди | 4×10–12 | Лопатки сводить в нижней точке |
| Тяга гантели одной рукой в упоре | 3×10–12/руку | Спина ровная, не скручиваться |
| Разведения гантелей в наклоне (rear delt) | 3×12 | Заменено с Face Pull |
| Сгибание бицепса гантелями стоя | 3×12 | Нейтральный хват |
| Bird-Dog | 3×12/сторону | С дыханием, контроль поясницы |
| Pallof Press в кроссовере | 3×12/сторону | Anti-rotation, ключевое для гиперлордоза |

---

## День 3 — ВЫХОДНОЙ (среда)

Сон на час дольше. Без активного восстановления.

---

## День 4 — UPPER (Верх тела + Кор)

| Упражнение | Подходы × Повторения | Примечания |
|---|---|---|
| Жим гантелей лёжа (или chest press тренажёр) | 4×8–10 | Нейтральная поясница |
| Тяга штанги в наклоне (или Т-гриф / тренажёр) | 4×10 | Спина нейтральная |
| Жим Арнольда сидя | 3×10 | Медленно, контролируемо |
| Подъём гантелей через стороны | 3×12 | Лёгкий вес |
| Rear delt fly | 3×12 | Наклон вперёд, локти чуть согнуты |
| Подъём ног в висе (колени к груди) | 3×10–12 | Без раскачки |
| Side Plank | 3×30–40 сек/сторону | С диафрагмальным дыханием |

---

## День 5 — LOWER (Ноги + Кор)

| Упражнение | Подходы × Повторения | Примечания |
|---|---|---|
| Гакк-присед или жим ногами (высокая постановка ног) | 4×10–12 | Поясница прижата к спинке |
| Румынская тяга гантелями | 3×10 | Лёгкий вес, нейтральная поясница |
| Выпады с гантелями (или болгарские сплит-приседания) | 3×10–12/ногу | Заменили ягодичный мостик со штангой |
| Сгибания ног в тренажёре | 3×12 | Изолирующее, безопасно для поясницы |
| Ab Rollout на коленях | 3×8–10 | Или в тренажёре, поясница нейтральная |
| Pallof Press в кроссовере | 3×12/сторону | Anti-rotation |
| Hollow Body Hold | 3×30–40 сек | С диафрагмальным дыханием |

> **Убрано:** Ягодичный мостик со штангой (неудобно технически), Обратная гиперэкстензия (риск усиления лордоза), Русские скручивания (заменены на Pallof Press)

---

## Ключевые принципы техники

- **Поясница:** нейтральная во всех упражнениях, никогда не прогибать дополнительно
- **Bracing:** до каждого рабочего подхода — вдох диафрагмой + лёгкое напряжение кора
- **Темп:** 2 сек вниз / 1 сек вверх / 1 сек пауза в пике
- **Отдых:** 60–75 сек между подходами
- **Жимы в одиночку:** только гантели или тренажёры, никогда не идти до отказа на штанге
- **Сигнал:** любое напряжение в пояснице → сразу снизить вес или заменить упражнение

## Прогрессия

Каждые 2 недели: +1–2 повторения **ИЛИ** +2–2,5 кг.
Когда все упражнения 4×12 с чистой техникой → переходим на следующий уровень.
